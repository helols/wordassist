Daum 에디터 - 개발 안내

서버 설정
  1. Apache와 Tomcat을 설치해주세요.
  2. Apache 셋팅
    1. Apache와 Tomcat을 연결하기 위해서는 jk_module을 사용합니다.
      mod_jk.so 파일을 APACHE_HOME/modules에 복사해 주세요.
    2. httpd.editor.conf 파일을 APACHE_HOME/conf에 복사해 주세요.
      httpd.editor.conf 파일을 열어 DocumentRoot를 해당 프로젝트 홈으로 수정해주세요.
      APACHE_HOME/conf의 http.conf 파일 하단에 다음을 추가해주세요.
      Include conf/httpd.editor.conf
  3. Tomcat 셋팅
    1. TOMCAT_HOME/conf의 server.xml을 열어 Host 아래에 www로 Context Path를 추가해주세요.
      <Context path="/editor" reloadable="true" docBase="D:\sources\XXX\www" workDir="D:\sources\XXX\work" />
    2. Eclipse에서 Tomcat Project>Update context definition을 이용하셔도 됩니다.
  4. build/build.properties의 WebApps properties를 환경에 맞게 수정해주세요.
※ v1.5 현재는 Apache, Tomcat, Velocity를 사용하는 환경만 지원합니다. 추후 다른 환경을 지원할 예정입니다.
에디터 실행
  1. 서버가 정상적으로 설정이 되면 www 아래에 있는 daumeditor.html을 실행시켜주세요.
    http://HOST/CONTEXT/daumeditor.html 에디터는 SOURCE, SNAPSHOT, RELEASE 세가지 모드로 실행시킬 수 있습니다.
  2. SOURCE 모드는 daumeditor 아래의 소스를 직접 참조합니다.
    daumeditor.html의 #loadResource("SOURCE") 부분을 수정하시면 됩니다.
    SOURCE 모드는 모든 리소스를 일일히 로드하여 사용하므로 서비스에 적용할 경우 매우 느릴 수 있습니다.
  3. SNAPSHOT 모드는 커밋된 소스를 체크아웃 받아 빌드하여 snapshot 아래에 배포하고 이를 참조합니다.
    daumeditor.html의 #loadResource("SNAPSHOT") 부분을 수정하시면 됩니다.
  4. RELEASE 모드는 커밋된 소스를 체크아웃 받아 배포 가능한 형태로 빌드하여 이를 참조합니다.
    새로운 버전을 생성하여 releases/VERSION 아래의 소스를 직접 참조합니다.
    daumeditor.html의 #loadResource("RELEASE") 부분과 $txVersion 부분을 수정하시면 됩니다.
    해당 디렉터리에 있는 폴더 또는 압축파일을 가지고 독립적으로 사용할 수 있습니다.
※ SNAPSHOT, RELEASE 일 경우에는 배포 과정이 필요합니다.
에디터 커스터마이즈
  1. daumeditor.html의 각 velocity macro 부분을 주석처리(##)하여 각 기능을 On/Off 할 수 있습니다.
    해당 macro는 WEB-INF/template/daumeditor/PartMacroCore.vm에서 확인하시면 됩니다.
  2. daumeditor.html의 new Editor({}) 부분의 옵션들을 수정하여,
    글자크기나 글자체, 글자색, 배경색 등 여러 스타일을 가진 에디터를 실행시킬 수 있습니다.
  3. 사진, 파일 첨부는 image.html, file.html를 참고하셔서 인터페이스를 맞추어 사용하시면 됩니다.
※ 자세한 사용법은 여기를 참고해주세요.
※ 사진, 파일 업로더, 자동저장의 Back-End 인터페이스는 추후 제공할 예정입니다.
빌드 및 배포
  1. 먼저 build/build.properties와 build/account.properties의 SVN properties 부분을 환경에 맞게 수정해주세요.
  2. SNAPSHOT 모드를 사용하기 위해서는 #_make_snapshot를 실행시켜 한꺼번에 빌드 및 배포를 할 수 있습니다.
    www/daumeditor 아래에 있는 소스를 통합하고 압축하는 과정을 거칩니다.
  3. RELEASE 모드를 사용하기 위해서는 #_make_release를 실행시켜 한꺼번에 빌드 및 배포을 할 수 있습니다.
    www/daumeditor 아래에 있는 소스를 통합하고 압축하고 패키징하는 과정을 거칩니다.
    배포 후에는 $editorMode를 SNAPSHOT로 설정하고 $txVersion을 높여 확인하면 됩니다.
